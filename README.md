# 📊 PhonePe Payment Insights Dashboard

A Power BI dashboard project built to analyze **PhonePe transaction data** and generate real-world business insights. This project covers complete **data cleaning, ETL, data modeling, DAX calculations, KPI creation, and interactive dashboard design**.

## 🚀 Key Features
- Transaction & Revenue Analysis  
- Payment Status Tracking  
- User Behavior Analysis  
- Age Group Contribution  
- Service-wise Transaction Analysis  
- Top Users Analysis  
- Weekday vs Weekend Usage  
- Interactive Filters & Tooltips  

## 🛠 Tools Used
- Microsoft Power BI  
- Power Query  
- DAX  
- Data Modeling  

## 📌 Key Insights
- Weekday transactions are higher than weekends  
- Millennials contribute the highest transactions  
- Loans and Money Transfers drive major revenue  
- Payment success rate remains high  

This project demonstrates practical skills in **Business Intelligence, Data Analysis, KPI Reporting, and Dashboard Storytelling**.
