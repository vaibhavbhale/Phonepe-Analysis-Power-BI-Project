## 📊 PhonePe Payment Insights Dashboard | Power BI

Developed an interactive **PhonePe Data Analysis Dashboard** in Microsoft Power BI to analyze payment trends, transaction behavior, and user engagement using real-world business scenarios. The project includes complete **data cleaning, ETL, data modeling, DAX measures, KPI tracking, and dashboard storytelling**.

This dashboard provides insights into **total transactions, transaction value, successful payment rate, user distribution by age groups, service-wise transaction analysis, top users, and weekday vs weekend usage patterns**.

### 🚀 Key Highlights

✔ Built end-to-end dashboard from raw data to insights
✔ Created KPIs for transactions, value, and success rate
✔ Designed interactive slicers for month and payment status
✔ Implemented custom tooltips for better user experience
✔ Performed age-segment and service-wise analysis
✔ Identified peak transaction periods and customer behavior
✔ Applied professional dashboard design and storytelling principles

### 📌 Business Insights

* Highest transactions observed on weekdays
* Millennials contribute the largest transaction share
* Loans and Money Transfer generate major transaction volume
* Payment success rate remains consistently high
* User engagement varies significantly by service type

### 🛠 Tools & Skills Used

* Microsoft Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization
* KPI Design
* Business Intelligence
